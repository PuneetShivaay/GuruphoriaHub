# **App Name**: Guruphoria

## Core Features:

- Landing Page: Display a hero section, mission statement, and featured courses.
- Course Listing: Display courses fetched from Firestore with thumbnails, descriptions, and a 'Watch Now' button.
- Video Player: Embed a video player with course details (title, instructor, duration, tags) and suggested videos.
- Dynamic Content: Use Firestore to store and retrieve course metadata (title, description, thumbnail, video link).
- User Authentication: Implement user sign-in with Google via Firebase Authentication (optional).
- Intelligent Recommendations: AI tool recommends videos and resources appropriate for the current lecture topic.

## Style Guidelines:

- Primary color: Deep blue-purple (#6759E9), inspired by the tech education theme, promoting a sense of trust and innovation.
- Background color: Light gray (#F4F3FA), a desaturated version of the primary, providing a clean, modern backdrop.
- Accent color: Cyan (#4ACDE6), an analogous color that creates contrast, highlighting calls to action and key information.
- Body font: 'Inter', sans-serif, modern, for clear and readable text. Headline Font: 'Poppins', sans-serif, for a contemporary feel.
- Use a consistent set of minimalist icons to represent course categories and functionalities.
- Maintain a clean, grid-based layout for course listings and video details to ensure readability and visual appeal.
- Subtle transition effects and loading animations to enhance user experience without being distracting.
